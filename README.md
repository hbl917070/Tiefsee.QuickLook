# Tiefsee4.QuickLook

此專案為 [Tiefsee4](https://github.com/hbl917070/Tiefsee4) 的擴充套件，<br>
在桌面或資料夾時，長按空白鍵即可透過 Tiefsee 來預覽檔案。
<br><br>
程式碼來自：https://github.com/QL-Win/QuickLook
